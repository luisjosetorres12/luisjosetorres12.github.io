OverTheWire es una plataforma que ofrece wargames (juegos de guerra) y retos para aprender ciberseguridad.

## Series Disponibles

- **[Bandit](/challenges/bandit/)** - Línea de comandos y conceptos básicos de Linux
- **Natas** - Vulnerabilidades web básicas
- **Leviathan** - Programación de sistemas y conceptos avanzados
- **Krypton** - Criptografía

---

## Cómo Usar Esta Sección

Cada serie tiene múltiples niveles. Para navegar:

1. Entra a la serie que quieras (ej: Bandit)
2. Sigue los niveles de menor a mayor dificultad
3. Usa los enlaces "Siguiente" y "Anterior" para navegar entre niveles
4. Los tags te permiten agrupar desafíos por concepto
